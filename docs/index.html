⚠️

June 2026: 400+ AUR packages were backdoored in a supply-chain attack — malicious PKGBUILDs silently fetching and running an infostealer. AUR packages are unsigned, community-maintained recipes built on your machine. aur-guard puts a gate in front of them.

A decision chain, per package

Every AUR update runs the gauntlet. Anything uncertain is delayed or blocked — never installed by default.

STEP 1

Whitelist

Trusted signed-binary repackagers skip the delay — but still get scanned & reviewed.

STEP 2

Delay (lag)

Installs the git revision from N days ago, exposed long enough for the community to flag it.

STEP 3

Anti-revert

Refuses a revision that was cleaned/reverted since — a backdoor lingers in git history.

STEP 4

Static scan

Delegates to aur-scan: 70+ rules and a known-IOC database.

STEP 5

AI review

An LLM reads the exact PKGBUILD diff and votes safe / suspicious before any build.

See the verdict, not the surprise

One command shows official updates, AUR updates, and a clear verdict for each — allowed, delayed, or blocked with a reason.

Built like a security tool

Fail-closed by design: on any doubt, it delays or blocks — it never silently downgrades or installs.

⏳

True deferred updates

Lag mode builds the exact git revision that was public N days ago — you always get updates, just safely behind.

🧠

AI review, multi-vote

Groq, Anthropic or OpenAI — configurable. One call when safe; extra confirmation votes only to back up a block.

🔑

One safe entry point

aur-guard upgrade does pacman -Syu then the guarded AUR — so you never bypass the gate.

🖥️

CLI, TUI & GTK4

Scriptable CLI, a keyboard TUI, and a polished libadwaita app with editable whitelist and suggestions.

🌍

Multilingual

gettext-based i18n following your system locale. English and French included, more easy to add.

🔐

Secrets stay secret

API keys come from the environment or a 0600 file — never written to config or logs.

Install

Requires Rust, an AUR helper (yay/paru) and — optionally — aur-scan for the static layer.

# clone & build git clone https://github.com/Xhelliom/aur-guard cd aur-guard cargo build --release --features gui # install binaries + translations install -Dm755 target/release/aur-guard ~/.local/bin/aur-guard install -Dm755 target/release/aur-guard-gui ~/.local/bin/aur-guard-gui po/install.sh # use it aur-guard check # report — installs nothing aur-guard upgrade # official repos + safe AUR
